📱 Jetpack Compose UI Practice

This project contains a collection of commonly used UI components built using Jetpack Compose.
The goal of this repository is to practice and demonstrate modern Android UI development patterns.

🚀 Features / Screens

🔹 Lists
	•	Vertical List – Built using LazyColumn
	•	Horizontal List – Built using LazyRow
	•	Grid Layout – Using LazyVerticalGrid

🔹 Dialogs
	•	Alert Dialog – Standard Material dialog
	•	Custom Dialog – Fully customizable UI using Dialog
	•	Horizontal Dialog Layout – Custom structured dialog design

🔹 Scrollable UI
	•	Vertical Scroll Screen – Using Column + verticalScroll
	•	Horizontal Scroll View – Using Row + horizontalScroll

🔹 ViewPager (Carousel)
	•	Basic ViewPager – Using HorizontalPager
	•	Custom ViewPager – Center-focused carousel with scaling and elevation effects

  🛠️ Tech Stack
	•	Kotlin
	•	Jetpack Compose
	•	Material 3
	•	Coil (AsyncImage)
	•	State Management (Compose State APIs)

  🎯 Purpose

This project is focused on:
	•	Understanding Compose layouts and modifiers
	•	Learning state-driven UI
	•	Building reusable and scalable UI components
	•	Practicing real-world UI patterns used in modern Android apps

  📌 Note
This is a practice project aimed at improving UI development skills in Jetpack Compose.
Code is kept simple and focused on understanding concepts rather than production-level architecture.
